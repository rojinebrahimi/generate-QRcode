# Generate QR Code
Script to generate QR code based on the given URL.

![github com](https://user-images.githubusercontent.com/44529498/113623772-0a69a300-9674-11eb-8fca-329b79a5f84b.png)

## Requirements
 * Python 3.8+
 * Pillow
 * qrcode

## Usage
 * Packages
 ```
 pip install qrcode[pil]
 ```
 * Running
 ```
 python QRcode.py
 ```
